# OrderAppAPI
This repository contains API's for Project

## Steps to run

1. `npm install`
2. `node index.js`
3. go to any client like postman or curl
4. api endpoints:
- [POST] /customer
- [POST] /order
- [POST] /shipment
- [GET] /shipment/{city}
- [GET] /allPurchaseOrders
- [GET] /allPurchaseShipmentOrders
